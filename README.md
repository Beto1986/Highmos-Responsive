# Highmos-Responsive

Web realizada basada en el sitio https://devchallenges.io/ en donde se trabaja con HTML, CSS y un poco de Javascript para activar la barra de navegación en una determinada cantidad de scrolls.(50px). Utiliza flex, grid, media querys y transiciones/animaciones para darle vida al sitio.  

Página de visita: https://beto1986.github.io/highmos-responsive/
